# GPIO计算方式

RK3399 GPIO

计算模板：

将引脚定义转换成以下形式 GPIOX_YZ 我们得到3个参数X、Y、Z；Y在引脚定义是以字母形式存在需要替换成数字，对应关系如下

```
A = 0
B = 1
C = 2
D = 3
```

算法：

(X*32)+(Y*8)+Z

计算示例：

```
GPIO1_A7   (1*32)+(0*8)+7 = 39
GPIO1_B1   (1*32)+(1*8)+1 = 41
GPIO2_C4   (2*32)+(2*8)+4 = 84
GPIO3_D0   (3*32)+(3*8)+0 = 120
GPIO3_D6   (3*32)+(3*8)+6 = 126
```
